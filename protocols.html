Internet Protocols
Internet protocol
Internet protocols are a set of rules for exchanging messages between computers (techopedia, 2019).

We have different types of internet protocol which are:

  • FTP (file transfer protocol)
  • Gopher
  • Telnet
  • HTTP (hypertext transfer protocol)
  • POP (Post office protocol)
  • Ethernet

FTP (file transfer protocol)

FTP is communication used for transferring files from the server to the client (wikipedia, 2021). Telnet

An application protocol is used to access a virtual computer over the internet and have communication (Fitzgibbons, 2021). It follows the user commands this protocol can use file transfer protocol (FTP) and HTTP.

How it works:

  • The user can send the commands to find which ports are open
  • Connect remotely to the virtual machine but need to enter their username and password
  • They can send files that using the FTP

The use of telnet:

  • The user can edit the files
  • Check emails
  • Run various applications
  • Play games

HTTP

The HTTP protocol is a protocol used to fetch documents such as HTML documents. It is the exchange of data between the web and the client meaning the requests are made by the user on the web.
