OVERVIEW OF THE INTERNET

The World Wide Web [WWW for short] are all the resources and users on the Internet that are using the Hypertext Transfer Protocol [HTTP] created by Tim Bernes-Lee in
1989. In simple terms, The World Wide Web is a way of exchanging information between computers on the Internet, tying them together into a vast collection of interactive
multimedia resources.

There is a commojn misconception that the Web and the Internet are the same thing. That is however untrue. The Web uses URL's [Uniform Resource Locators] to identify
documents accesible over the internet. This information is transferred via Hypertext Transfer Protocol and is accessed via web browsers and published on web servers.
